# Personal Website

Hey there! This is the codebase for my personal website, which can be found at [rosti.no](https://rosti.no). It's built using [Next.js](https://nextjs.org/), styled with [styled-components](https://styled-components.com/) and deployed to [Vercel](https://vercel.com/).

This is just a simple website that contains my contact info.

One cool feature of the website is the Spotify card that shows what music I'm currently listening to.

I also implemented a shuffle for the contact cards depending on the visitor's source. So if you're coming from LinkedIn, Glassdoor, or something similar, the order and visibility of the cards will be shuffled.
